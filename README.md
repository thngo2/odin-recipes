# odin-recipes
Building a basic recipes site from the Odin Project

# Technologies
- VSCode
- HTML

# Objective
The objective of this project is to use HTML to learn:
[x] How to use unordered and ordered lists
[x] How to create links to pages on other websites on the internet (absolute links)
[x] How to create links to other pages on your own websites (relative links)
[x] How to display images
[x] How to use the alt attribute to describe images
[x] How to work with text (bold, paragraphs, headings, italicize)
[x] Parent directories 
[ ] Comment my code

# Reflection
The Odin Recipe site was a simple but fun build. I was able to complete six out of the seven objectives and I also learned how to center my site. While building this site, I thought about linking my recipe files together and decided to make all my recipes have the "Odin Recipe" title so that the user was able to easily return to the homepage. This feature and adding my images improved my understanding of parent directories and the importance of ./ and ../. One thing I forgot to do was comment on my code. Overall, this project is a fun start to learning how code in HTML!
